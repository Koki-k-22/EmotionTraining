
# 国語の「心情把握問題」の作問技法  
—Active Listening練習アプリへの転用を目的とした調査レポート—

## 0. 調査上の注意

以下では、記述を次のように区別する。

- **［事実］** 公的資料、実際の入試・学力調査、学術研究に明記されている内容
- **［整理］** 複数資料に共通する実務知を、本レポートで分類・一般化したもの
- **［提案］** Active Listeningアプリ向けの設計案。入試機関が公式に示した基準ではない

また、大学入学共通テストの国語記述式問題は試行調査まで行われたが、2021年1月の本試験への導入は見送られた。したがって、記述式採点については、試行調査・導入検討資料、全国学力・学習状況調査、個別入試・教育測定研究を参照する。[文部科学省「記述式問題の導入見送り」](https://www.mext.go.jp/magazine/backnumber/1422844_00003.htm)

# 1. 心情把握問題の典型的な設問形式

## 1.1 基本類型

| 類型 | 典型的な設問文 | 主に測るもの |
|---|---|---|
| 直接的心情説明 | 「このときの○○の気持ちを説明しなさい」 | 心情の同定、言語化 |
| 選択式心情説明 | 「傍線部における○○の心情として最も適当なものを選べ」 | 複数の解釈の比較 |
| 行動理由 | 「○○がこのような行動をしたのはなぜか」 | 行動から動機・感情への推論 |
| 発話意図 | 「○○がこのように言ったのはなぜか」 | 表面上の発話と本音の区別 |
| 表情・態度の解釈 | 「『目を伏せた』とあるが、ここにはどのような思いが表れているか」 | 非言語的手掛かりの解釈 |
| 比喩の内容説明 | 「この比喩は○○のどのような心情を表しているか」 | 表現と心情の対応 |
| 情景描写の効果 | 「この情景描写は主人公の心情とどのように関わるか」 | 間接的・象徴的表現の理解 |
| 心情変化 | 「○○の気持ちは、何をきっかけに、どのように変化したか」 | 前後差、転機、因果 |
| 複合感情 | 「○○がうれしいだけではいられなかったのはなぜか」 | 葛藤・両価感情 |
| 人物関係 | 「このとき○○は△△をどのような人物だと捉えているか」 | 相手についての認識と感情 |
| 根拠選択 | 「心情を読み取る根拠となる表現を選べ／抜き出せ」 | 解釈と本文根拠の接続 |
| 心情語補充 | 「空欄に入る心情を表す語として適切なものを選べ」 | 心情語彙と文脈理解 |
| 読みの比較 | 「AさんとBさんの解釈のうち、本文に即しているものを選べ」 | 解釈の妥当性評価 |
| 表現生成 | 「この心情が伝わるよう、情景・行動を用いて続きを書け」 | 心情を描写へ変換する力 |

中学入試の出題者資料でも、文学的文章の中心を「心情の変化」とし、表情、態度、言動、情景描写、比喩、言動に至った理由を手掛かりとして挙げている。[吉祥女子中学校「2025年度入試問題 問題構成・出題の意図」](https://www.kichijo-joshi.jp/wp/wp-content/uploads/2025/07/647ba193369ab43e4f49aa4297af3c11.pdf)

文部科学省の学習指導要領解説も、人物像を捉えるには、登場人物の行動・会話・様子を表す「複数の叙述」を結び付け、性格や考え方を総合的に判断する必要があるとしている。[文部科学省「小学校学習指導要領解説 国語編」](https://www.mext.go.jp/content/20220606-mxt_kyoiku02-100002607_002.pdf)

## 1.2 学校段階による傾向

**［整理］**

- 小学校・中学入試では、行動、会話、表情、場面変化など比較的観察可能な情報が中心になる。
- 高校入試では、心情変化のきっかけ、人物関係、比喩や情景の効果が加わる。
- 大学入試では、局所的な心情だけでなく、作品全体における人物認識、語り手の自己理解、矛盾した感情、表現構造との関係まで問われやすい。
- 共通テストでは、単純に「悲しい／うれしい」を当てるだけでなく、会話における態度の背後、動作描写、人物への認識を別々に問う構成が見られる。

2024年度共通テスト国語第2問「桟橋」について、問題作成部会は、問3を「友人の態度の背後にある心情」、問4を「糸屑を拾う動作描写が表す内容」、問5を「主人公のおばに対する認識と思い」と説明している。[大学入試センター「令和6年度国語・問題作成部会の自己評価」](https://www.dnc.ac.jp/albums/abm.php?d=677&f=abm00004551.pdf&n=R6%E6%9C%AC%E8%A9%A6%E9%A8%93_%E5%9B%BD%E8%AA%9E_%E8%87%AA%E5%B7%B1%E8%A9%95%E4%BE%A1.pdf)

# 2. 正答の根拠を本文に埋め込む方法

## 2.1 心情の手掛かりとなる情報

### 直接的な手掛かり

- 「悔しかった」「安心した」などの心情語
- 内的独白
- 心情を説明する語り手の叙述
- 「～と思った」「～を恐れた」などの認知・評価

直接表現は正答を安定させやすい一方、そのまま質問すると単純な抜き出し問題になる。通常は、心情語を別の表現に言い換えさせたり、その理由・対象・きっかけまで問う。

### 間接的な手掛かり

- **行動**：手を止める、物を乱暴に置く、何度も確認する
- **表情・身体反応**：目を伏せる、頬が緩む、声が震える
- **発話内容**：婉曲表現、否定、話題転換、過度な冗談
- **発話形式**：沈黙、言い直し、繰り返し、語尾の変化
- **人物関係**：親密度、上下関係、過去の対立、期待
- **前後の出来事**：期待の成立、失敗、誤解の解消
- **情景描写**：天候、光、音、色、空間の広がり
- **比喩・象徴**：胸につかえる、視界が開ける、距離を感じる
- **対比**：以前は話していたのに急に黙る
- **反復**：同じ言葉・動作が心情変化の前後で意味を変える

学習指導要領に基づく指導資料では、心情は直接書かれるだけでなく、人物関係に基づく行動、会話、情景を通して暗示される場合があると説明されている。[国立教育政策研究所「令和2年度 全国学力・学習状況調査 調査問題活用資料」](https://www.nier.go.jp/sankou/r02/data/20p.pdf)

## 2.2 作問原則のチェックリスト

以下は、入試・指導資料をActive Listening課題へ転用した設計チェックリストである。

### A. 先に正答を定義する

- [ ] 問いたい人物・時点・対象を一つに固定したか
- [ ] 中核心情を1～2語で定義したか
- [ ] 心情の強度を定義したか  
  例：不安／強い不安／絶望
- [ ] 何に対する感情かを定義したか
- [ ] きっかけとなった出来事を定義したか
- [ ] その人物の認識・解釈を定義したか  
  例：「失敗した」だけでなく、「自分は期待に応えられなかったと捉えた」
- [ ] 複合感情なら主感情と副感情を分けたか

推奨する内部表現は次の連鎖である。

> 出来事 → 人物の認識・評価 → 感情 → 欲求・目標 → 発話／行動

たとえば、

> 上司が返事をしなかった  
> → 自分の提案を評価していないと受け取った  
> → 不安と落胆  
> → 評価を確認したいが、しつこいと思われたくない  
> → 「まあ、急ぎではないので」と引き下がる

この連鎖を先に作ると、行動と正答の因果が逆転しにくい。

### B. 根拠を複数配置する

- [ ] 正答を支持する手掛かりが少なくとも2個あるか
- [ ] 近接手掛かりと文脈手掛かりを一つずつ置いたか
- [ ] 異なるチャネルの手掛かりを組み合わせたか  
  例：発話＋沈黙、行動＋過去の出来事
- [ ] 手掛かり同士が矛盾せず、同じ心情に収束するか
- [ ] 単一の感情語だけを見れば解ける問題になっていないか
- [ ] 正答のすべての構成要素に本文根拠があるか

### C. 表面発話と深層心情を区別する

- [ ] 社交辞令、婉曲、冗談、否定など、表面と本音を分ける仕掛けがあるか
- [ ] ただし「発話と本音は必ず逆」とする単純な裏読みになっていないか
- [ ] 本音を推測できる補助根拠があるか
- [ ] 「怒っているように見えるが、実は悲しい」のような反転には、十分な文脈があるか

### D. 心情変化を問う場合

- [ ] 変化前と変化後がそれぞれ本文に示されているか
- [ ] 転機となる出来事が特定できるか
- [ ] 変化の方向だけでなく、理由も説明可能か
- [ ] 心情曲線上のどの時点を問うか明確か
- [ ] 「不安から安心」のような単純変化か、「期待しつつ不安」のような複合化かを決めたか

### E. 曖昧さを制御する

- [ ] 複数解釈が可能な文学的曖昧さと、採点不能な情報不足を区別したか
- [ ] 別解を排除する根拠、または別解を部分点にする基準があるか
- [ ] 一般常識だけで正解できず、本文確認が必要か
- [ ] 読者自身ならどう感じるかではなく、その人物がどう受け取ったかを問うているか
- [ ] 人物の内心を事実として断定しすぎていないか

最後の点はActive Listeningアプリでは特に重要である。実際の人間の内面は一意に確定できないため、設問も「本当の気持ちは何か」より、

> 「この発話と行動から、最も強くうかがえる気持ちは何か」

とする方が妥当である。

# 3. 誤答肢設計のパターン集

## 3.1 一般的な選択肢作成原則

教育測定の一般原則では、誤答肢は、

- 明確に誤りである
- しかし未熟な理解を持つ受験者にはもっともらしい
- 正答と同じ種類・抽象度である
- 文法・長さ・具体性などの表面的特徴から正答が分からない

ことが望ましいとされる。[Haladyna & Downing, “Multiple-Choice Item-Writing Guidelines”](https://testing.byu.edu/handbooks/Multiple-Choice%20Item%20Writing%20Guidelines%20-%20Haladyna%20and%20Downing.pdf)

選択肢の内容を同質にすると識別力が改善するという研究もあるが、すべての項目に無条件で適用できるほど証拠が確定しているわけではない。[“The Effect of Option Homogeneity in Multiple-Choice Items”](https://pmc.ncbi.nlm.nih.gov/articles/PMC6376538/)

なお、「言い過ぎ」「本文にない」「因果のすり替え」といった名称は受験指導で広く使われる整理であり、大学入試センターが公式の誤答肢生成マニュアルとして公表している分類ではない。

## 3.2 心情問題に特有の誤答パターン

| パターン | 作り方 | 例 | アプリ上の評価 |
|---|---|---|---|
| 程度の逸脱 | 感情の方向は合うが強すぎる／弱すぎる | 「不安」→「人生に絶望している」 | △ |
| 対象のずれ | 感情は合うが誰・何に対するものかが違う | 同僚への苛立ち→自分への怒り | △～× |
| 主体のずれ | 別人物の感情を入れる | 話し手の不安→聞き手の不安 | × |
| 時点のずれ | 前場面または後場面の心情を使う | 安心後の気持ちを直前に当てる | △～× |
| 部分正解 | 副次的心情だけを述べる | 罪悪感＋安堵のうち「安堵」のみ | △ |
| 核心欠落 | 感情の方向だけで理由・対象がない | 「悲しい」だけ | △ |
| 本文にない動機 | もっともらしい背景を追加する | 過去に裏切られたから疑っている | × |
| 因果の逆転 | 感情と行動の因果を逆にする | 不安だから黙った→黙ったので不安になった | × |
| 因果のすり替え | 本文中の別の出来事を原因にする | 評価への不安→予定変更への不満 | △～× |
| 表面発話への固着 | 言葉を字義通りに受け取る | 「大丈夫」→本当に心配していない | △ |
| 過剰な裏読み | 明示的発話を根拠なく反転する | 「うれしい」→本当は嫌っている | × |
| 一般論への逃避 | 人物固有の心情を常識に置換する | 「誰でも失敗すれば悲しい」 | △ |
| 人物評価との混同 | 感情ではなく性格を答える | 「慎重な人だ」 | △ |
| 事実の要約止まり | 出来事だけ述べ心情を述べない | 「返事をもらえなかった」 | △ |
| 感情ラベルだけ一致 | 「不安」は合うが本文と違う理由を付ける | 将来が不安→相手に嫌われるのが不安 | △～× |
| 反対感情 | 本文と感情極性が逆 | 安堵→失望 | × |
| 過度な断定 | 可能性を確定事実化する | 「嫌われたかもしれない」→「嫌われたと確信」 | △ |
| 道徳的評価の混入 | 心情ではなく善悪を答える | 「反省すべきだと思っている」 | △～× |
| 願望と感情の混同 | 目標・希望だけ述べる | 「仲直りしたい」だけ | △ |
| 矛盾混入 | 選択肢の前半は正しいが後半が本文と矛盾 | 心配しているが、もう関わりたくない | × |

## 3.3 「○許容／△ズレ」の勾配設計案

**［提案］** 二値正誤より、次の段階がActive Listeningの訓練に適する。

| 段階 | 定義 | 例 |
|---|---|---|
| ○ 中核一致 | 心情、対象、理由、強度が本文に整合 | 「期待に応えられなかったと思い、申し訳なく感じている」 |
| ○ 許容言換え | 同じ心理状態を自然な別語で表現 | 申し訳なさ→罪悪感／後ろめたさ |
| ○寄り△ | 中核心情は一致するが、理由・対象が省略 | 「申し訳なく感じている」 |
| △ 部分一致 | 副次的感情のみ、または感情の方向のみ一致 | 「落ち込んでいる」 |
| △ 意味近接 | 心情は近いが強度・ニュアンスがずれる | 不安→恐怖、残念→絶望 |
| △ 文脈ずれ | 場面全体にはあるが、問われた時点では中心でない | 不安解消後にも「不安」と答える |
| △ 根拠不足 | あり得るが本文から十分支持されない | 「相手に嫉妬している」 |
| × 構造誤り | 主体、対象、因果、時間を取り違える | 自分への怒りを相手への怒りとする |
| × 矛盾 | 本文の明示情報に反する | 安心している人物を絶望とする |

### 誤答肢を難易度順に作る方法

- 初級：反対感情、別人物、明白な本文外情報
- 中級：部分正解、強度の逸脱、表面発話への固着
- 上級：中核心情は正しいが対象がずれる、理由だけが違う、複合感情の一方を落とす
- 最上級：本文の一部分には合うが、作品全体・人物関係・時間軸には合わない

一問の選択肢をすべて異なる誤り方にすると、誤答ログ自体が診断情報になる。

# 4. 記述式回答の要素分解と採点

## 4.1 入試・学力調査に見られる要素採点

全国学力・学習状況調査では、正答条件を複数の要素に分け、その充足の組合せを「解答類型」として分類する方式が採られている。たとえば、比喩表現に関する設問では、

1. 該当する比喩表現を取り上げる
2. 誰・何のどのような様子かを明示する
3. 感じたこと・考えたことを具体的に書く

という条件に分解されている。[広島県立教育センター「国語科の評価問題における記述式問題の採点過程に関する研究」](https://www.pref.hiroshima.lg.jp/uploaded/attachment/615941.pdf)

同研究は、適切な採点基準について、

1. 測定対象となる知識・能力を明確化する
2. 能力間の構造と重みを整理する
3. 各観点を具体的な到達段階として言語化する
4. 軽重に応じて配点する
5. 複数採点者でプレ採点し、ずれを基に基準を修正する

ことを提案している。基準文は「適切に書いている」のような抽象語ではなく、「○○を明示している」のように、採点者の斟酌が入りにくい形にすることが推奨されている。

共通テスト記述式の検討資料でも、内容確認の例として「結論」「根拠」「理由」が挙げられ、第1次基準、採点者用のより精緻な第2次基準、各大学の第3次基準という階層が検討されていた。[文部科学省「記述式のセンター・大学共同採点の場合の主な論点」](https://www.mext.go.jp/content/20191224-mxt_daigakuc02-000003534_2.pdf)

## 4.2 心情記述用の採点要素

**［提案］** 心情問題は次の5要素に分解できる。

| 要素 | 内容 | 配点例 |
|---|---|---:|
| E：Emotion | 中核となる心情・複合感情 | 0～3 |
| T：Target | 誰・何に対する感情か | 0～1 |
| C：Cause/Trigger | きっかけ、状況、人物の認識 | 0～2 |
| R：Rationale | 発話・行動・文脈との結び付き | 0～3 |
| M：Modality | 程度、確信度、両価性 | 0～1 |
| 合計 |  | 10 |

### 採点例

設問：

> 「まあ、別に急ぎじゃないから」と視線を落とした話し手の気持ちを説明してください。

正答モデル：

> 返事をもらえず、自分の相談が相手に重く受け取られたのではないかと不安になっているが、迷惑を掛けたくないため、その不安を隠して引き下がろうとしている。

評価：

- E：不安＋遠慮＝3
- T：相手との関係について＝1
- C：返事がない／重く受け取られたという認識＝2
- R：「急ぎではない」「視線を落とす」と結び付く＝3
- M：断定ではなく「～ではないか」＝1

### 部分答案

> 「相手に迷惑を掛けたくなくて遠慮している。」

- E：遠慮＝2
- T：相手＝1
- C：迷惑を掛けたくない＝1
- R：本文との関係が暗黙的＝1
- M：0  
合計5点程度

### 上限制約

次の誤りは、要素点の単純加算だけでは過大評価されるため、上限を設ける。

- 主体を取り違えた：最大2点
- 心情の正負が逆：最大2点
- 本文と明白に矛盾：原則0点
- 本文にない出来事を核心的理由にした：最大3点
- 感情を述べず事実要約だけ：最大3点

## 4.3 採点の信頼性を高める運用

- 正答例を一つだけでなく3～5種類用意する
- 同義語辞書を作る  
  例：申し訳なさ／罪悪感／後ろめたさ
- 境界答案を用意する  
  ○下限、△上限、△下限、×
- 実回答を50～100件見てから基準を改訂する
- 人手採点の一致率を確認する
- 不一致が多い要素は、基準か問題文を修正する
- 感情語の完全一致ではなく、意味単位で採点する

記述式採点研究では、採点者間の差は、明確な主張よりも「理由」や言語表現で生じやすいことが報告されている。また、基準を協議した後でも差が残るため、解答例を用いた訓練と基準の継続的修正が必要である。[広島県立教育センター研究](https://www.pref.hiroshima.lg.jp/uploaded/attachment/615941.pdf)

# 5. 心情理解に関する教育・心理研究

## 5.1 Theory of Mindとの関係

Theory of Mind（ToM）は、他者が自分とは異なる信念、欲求、意図、感情、知識を持つことを理解し、その心的状態を推論する能力を指す。

2024年公表のメタ分析では、47の独立標本、5,123人、157効果量を統合した結果、ToMと文章理解の間に有意な関連が認められた（相関 `r = .33`）。言語能力や実行機能を統制しても関連は残った（`r = .22～.32`）。ただし、これは関連を示す結果であり、「心情問題を解けばToMが必ず向上する」という因果を直接証明するものではない。[Tompkins et al., “Theory of mind and text comprehension across the lifespan”](https://pubmed.ncbi.nlm.nih.gov/39509197/)

**［提案］** Active Listening課題では、単なる感情ラベル当てより、

- 話し手が何を知っているか
- 何を誤解しているか
- 何を期待しているか
- 相手がどう受け取ると思っているか
- 表面発話と内心がなぜ異なるか

まで追跡する課題の方が、ToMに近い訓練になる。

## 5.2 「叙述に基づく推論」の重要性

描写を具体的な情景としてイメージさせることと、心情解釈の根拠となる文章を探して妥当性を検証させることが、物語文の心情解釈を改善する教授方略として研究されている。[「描写の映像化と解釈の検証が物語文の読みに及ぼす効果」](https://www.jstage.jst.go.jp/article/japtl/1/2/1_KJ00007505122/_article/-char/ja/)

一方、人物への親密度が心情判断に影響するという研究もあり、読者自身の対人経験が推論に入り込む可能性が示されている。[「物語文の理解を促進する人物の親密度」](https://www.jstage.jst.go.jp/article/pacjpa/84/0/84_PI-086/_article/-char/ja)

したがって、アプリでは、

> 「あなたならどう感じるか」

と、

> 「この話し手について、会話から最も支持される解釈は何か」

を分ける必要がある。

## 5.3 心情曲線

心情曲線は、少なくとも1950年代には折れ線グラフによる感情変化の表現が紹介され、1960～70年代には「心情曲線」「心理曲線」「感情曲線」などの名称で用いられていた。[国立国会図書館レファレンス協同データベース「心情曲線の来歴」](https://crd.ndl.go.jp/reference/entry/index.php?id=1000345488&page=ref_view)

教育実践では、縦軸にプラス／マイナスの心情を置き、場面ごとの変化を表す方法が使われている。[教職員支援機構「小4国語科実践事例」](https://www.nits.go.jp/jisedai/achievement/jirei/jirei200.html)

ただし、正負一軸だけでは、

- うれしいが不安
- 腹が立つが申し訳ない
- 寂しいが解放感もある

といった複合感情を表しにくい。

**［提案］** アプリでは一軸ではなく、

- 快―不快
- 覚醒度
- 接近―回避
- 確信―不確実
- 主感情／副感情

を分けた「多次元心情曲線」が適する。

# 6. 良問・参考問題の実例

## 例1：2024年度大学入学共通テスト「桟橋」

第2問では、同じ人物関係を三つの異なる観点から問うている。

- 問3：友人の会話態度から、背後の心情を推論
- 問4：主人公が糸屑を拾う動作から、心情・心理状態を推論
- 問5：主人公が「おば」をどう認識し、何を感じたかを統合

問題作成部会はこれらを「読解力の力量を測る適切な問題」と評価している。また、素材文自体も、主人公の内面描写が丁寧で、心情変化を把握する文学的文章として適切だったとしている。[問題冊子・正解への公式入口](https://www.dnc.ac.jp/kyotsu/kakomondai/r6/r6_honshiken_mondai.html)、[問題作成部会自己評価](https://www.dnc.ac.jp/albums/abm.php?d=677&f=abm00004551.pdf&n=R6%E6%9C%AC%E8%A9%A6%E9%A8%93_%E5%9B%BD%E8%AA%9E_%E8%87%AA%E5%B7%B1%E8%A9%95%E4%BE%A1.pdf)

**アプリへの示唆：** 一つの会話に対し、「発話」「微細な行動」「相手への認識」の三層を別問題にできる。

## 例2：2024年度全国学力調査「オニグモじいさん」

児童が心に残った表現と理由を書く問題で、正答例には、

- 発話から、相手を傷つけまいとする温かさを読むもの
- 小鳥の歌という情景から、人物のわずかな喜びを読むもの

が示された。正答率は72.7％で、行動・会話・様子を基に人物像や物語全体を想像する問題として分析されている。[国立教育政策研究所「令和6年度調査結果概要」](https://www.nier.go.jp/kaihatsu/setsumeikai/r06setsumeikai/24ers.pdf)

**アプリへの示唆：** 同じ心情について、直接的な発話根拠と間接的な情景根拠の双方を正答として認められる。

## 例3：2024年度全国学力調査「紙の辞書」

「次の出番への期待」という心情が伝わるように物語の続きを描写し、その表現の効果を説明させる問題である。正答例では、窓から差し込む光を使って期待感を表現している。正答率は49.8％だった。[国立教育政策研究所「物語を創作する—紙の辞書」](https://www.nier.go.jp/24chousakekkahoukoku/report/data/24mlang_idea_03.pdf)

**アプリへの示唆：** 心情を「当てる」だけでなく、与えられた心情をどの発話・沈黙・行動で表現するかを選ばせる逆向き課題が作れる。

# 7. Active Listeningアプリへの実装提案

## 推奨する一問の内部データ構造

```yaml
speaker: A
target_time: turn_8
target_person: A
surface_utterance: "まあ、急ぎじゃないから"
observable_behavior:
  - 視線を落とす
  - 直前まで返事を待っていた
trigger:
  - 相手が返答しなかった
appraisal:
  - 自分の相談は迷惑だったかもしれない
primary_emotion:
  - 不安
secondary_emotion:
  - 落胆
social_motive:
  - 相手に負担を掛けたくない
response_tendency:
  - 本題から引き下がる
evidence:
  proximal:
    - "急ぎじゃないから"
    - 視線を落とす
  contextual:
    - 返事を待っていた
acceptable_answers:
  - 不安と遠慮
  - 迷惑だったのではないかという心配
partial_answers:
  - 落ち込んでいる
  - 遠慮している
unsupported_answers:
  - 怒っている
  - 相手を嫌っている
```

## 最終品質チェック

- [ ] 心情の正解を本文より先に決めた
- [ ] 正解を支える独立した手掛かりが2個以上ある
- [ ] 発話だけでなく、文脈・行動・関係性を使っている
- [ ] 主体、時点、感情の対象が明確
- [ ] 正答に本文外の過去や性格を要求していない
- [ ] 表面的発話と深層心情の差に根拠がある
- [ ] 複合感情を単一の正負ラベルに潰していない
- [ ] 誤答肢は実際に起こりそうな読み違いから作った
- [ ] 誤答肢の長さ・文体・具体性が正答とそろっている
- [ ] 正答だけが慎重で、誤答だけが極端な表現になっていない
- [ ] ○、△、×の境界答案が用意されている
- [ ] 実回答によるプレテストを行う
- [ ] 正答率だけでなく、どの誤答パターンが選ばれたかを確認する
- [ ] 「相手の本心を断定する能力」ではなく「証拠に基づく仮説形成」を評価している

# 8. 結論

国語の心情問題から最も有効に転用できる原則は、心情語の当てっこではなく、

> **複数の叙述を統合し、人物が出来事をどう受け取ったかを推論し、その解釈を根拠とともに説明させる**

という構造である。

Active Listening用には、正答を「唯一の本心」と扱うのではなく、

1. 本文から強く支持される中核解釈
2. 許容できる言い換え
3. 部分的だが核心を欠く解釈
4. あり得るが根拠不足の解釈
5. 文脈と矛盾する解釈

という勾配で評価するのが望ましい。

特に良質な問題は、単一の感情語ではなく、

> 出来事 → 認識・評価 → 複合感情 → 対人的欲求 → 発話・行動

の連鎖を復元させる。この構造なら、国語入試の読解技法を、現実の傾聴に必要な「断定せず、根拠を持って相手の内面を仮説化する力」へ比較的自然に接続できる。
